### 2. ResNet50/ResNet101

**代码位置**: [yolo_classifier.py#L52-82](file:///f:/04-CODE/webkeshe/ai-service/classifiers/yolo_classifier.py#L52-L82)

**作用**: 深度特征提取器，提取2048维特征向量

**实现代码**:
```python
class FeatureExtractor:
    def __init__(self, model_name='resnet50'):
        self.model = models.resnet50(weights=models.ResNet50_Weights.IMAGENET1K_V1)
        self.model = nn.Sequential(*list(self.model.children())[:-1])  # 去掉最后一层
        self.feature_dim = 2048
    
    def extract(self, image):
        img_tensor = self.transform(image).unsqueeze(0).to(self.device)
        with torch.no_grad():
            features = self.model(img_tensor)
        return features.cpu().numpy().flatten()  # 返回2048维特征
```

**特点**:
- 使用预训练权重（IMAGENET1K_V1）
- 去掉全连接层，只保留卷积特征提取部分
- 输出2048维特征向量

```python
class SkinDetector:
    @staticmethod
    def detect(image):
        # YCbCr颜色空间
        ycbcr = cv2.cvtColor(image, cv2.COLOR_BGR2YCrCb)
        mask_ycbcr = cv2.inRange(ycbcr, [0,133,77], [255,173,127])
        
        # HSV颜色空间
        hsv = cv2.cvtColor(image, cv2.COLOR_BGR2HSV)
        mask_hsv = cv2.inRange(hsv, [0,20,40], [25,255,255])
        
        return cv2.bitwise_or(mask_ycbcr, mask_hsv)  # 融合两个mask
```

**分形维数计算**（[yolo_classifier.py#L166-182](file:///f:/04-CODE/webkeshe/ai-service/classifiers/yolo_classifier.py#L166-L182)）:
```python
def _fractal_dimension(contour):
    perimeter = cv2.arcLength(contour, True)
    area = cv2.contourArea(contour)
    dimension = (4 * np.pi * area) / (perimeter ** 2 + 1e-10)
    dimension = 2 - np.log(dimension) / np.log(max_dim / 10)
    return max(1.0, min(2.0, dimension))  # 范围1.0-2.0
```

**评分计算**（[yolo_classifier.py#L433-444](file:///f:/04-CODE/webkeshe/ai-service/classifiers/yolo_classifier.py#L433-L444)）:
```python
def _calculate_blood_score(self, blood_result):
    blood_ratio = blood_result.get('blood_ratio', 0.0)
    fractal_dim = blood_result.get('fractal_dimension', 0.0)
    
    base_score = min(blood_ratio * 400, 60)
    if fractal_dim > 1.4:
        base_score *= 1.6  # 分形维数高则评分提升
    
    return min(base_score, 80)
```

### 7. Hu矩

**代码位置**: [yolo_classifier.py#L377-380](file:///f:/04-CODE/webkeshe/ai-service/classifiers/yolo_classifier.py#L377-L380)

**作用**: 提取形状不变矩特征，用于形状识别

**实现代码**:
```python
moments = cv2.moments(skin_mask)
hu_moments = cv2.HuMoments(moments)  # 7个不变矩
hu_moments = -np.sign(hu_moments) * np.log10(np.abs(hu_moments) + 1e-10)
features.extend(hu_moments.flatten().tolist())  # 加入特征向量
```

**Hu矩特点**:
- 7个不变矩，对旋转、缩放、平移具有不变性
- 用于描述形状特征
- 归一化处理：取对数并保持符号

## 三、技术栈作用总结

| 技术栈 | 作用 | 输出 | 权重（正常帧） | 权重（血腥帧） |
|--------|------|------|---------------|---------------|
| **ViT-Large** | NSFW图像分类 | is_nsfw + confidence | 50% | 35% |
| **YOLOv8** | 敏感物体检测 | objects + risk_score | 15% | 10% |
| **SVM** | 特征分类 | NSFW概率 | 15% | 15% |
| **肤色检测** | 皮肤区域识别 | skin_ratio | 辅助 | 辅助 |
| **血腥检测** | 血液区域识别 | is_blood + blood_ratio | 10% | 30% |
| **Hu矩** | 形状特征提取 | 7维特征 | SVM输入 | SVM输入 |
| **ResNet** | 深度特征提取 | 2048维特征 | 可选 | 可选 |
